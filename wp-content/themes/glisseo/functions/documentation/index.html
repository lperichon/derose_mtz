Install

This chapter will give you detailed instructions on how to install WordPress and setup the theme with all its options and demo content.

0. Quick Agenda

  • (Install WP)
  • Download Trend from Themeforest
  • Unzip downloaded file
  • Install trend.zip inside via WP Theme Installer
  • Install revslider.zip (in folder revslider) inside via WP Theme Installer (see chapter slider later in this documentation)
  • Activate
  • Read "First Steps"
     
  • All what is missing in your theme now are sidebar widgets!

If you have a menu already in use please check that it is named "Main"! (Important!)

 

Yeah, you uploaded and activated our Theme! Thanks and good work so far!

The following steps are necessary or optional to get the Theme on the road for its first time:

  1. Import Example Content (optional):
    If you like some example content to see how things are done please go to the WP menu "Tools" | "Import" and choose WordPress.



    You will be asked to install the WP Import plugin if you not have done already. Please do so.



    Now import our Example data by choosing the "trend_example.xml" we have put in the "democontent" folder of our theme (should be there in the folder you have of our unzipped theme on your harddrive). Important is to import the media files too.

  2. Set Reading Pages:
    Please go to the corresponding page via the WP menu on the left side. It is "Settings" | "Reading".
    Choose "A static page" for frontpage display. After that you can choose our example "home" for frontpage and the "blog" page for blog (or the pages of your choice). Then press the "Save Changes" button there.



    Be sure that the pages you choose have the corresponding "page template" ("home-page" and/or "blog").
  3. Featured Images:
    And do not forget that you need to set featured Images to see the theme at it's full potential!

  4. Plugins:
    For optimal use of this plugin you should also install at least the Revolution Slider Plugin for the nice sliders and the Contact Form 7 plugin for using contact forms. Both installations are explained later in this document.

Trend Options

Options Overview

The Trend custom options panel can be found in the WordPress admin under the "Trend" tab. Options are separated into different categories to give you a better overview over all available options.

Note that when making any changes to the options, you need to click "Save All Changes" at the top or the bottom of the Trend options panel.

  • Highlight

    There is one highlight color throughout the theme (in the demo it is a green tone). You can change it here.

  • FavIcons

    Since this a responsive Theme which is build to work with mobile devices as well you can upload several favicons in different sizes here for all the different purposes mobile devices have for them. This fields are optional. And there is of course the standard favicon ico for desktop machines.

  • Header

    All things in the header can be defined here like the Site logo, backgrounds, heights and Typography.

  • Body

    All important things in the body area can be defined here like colors, backgrounds, heights and Typography.

  • Footer

    Turn the footer on or off and define colors , backgrounds, heights and Typography.

  • SubFooter

    Turn the sub footer on or off and define the sub footers content.

Widgets

Available Custom Widgets

  • Trend Archives

    This widget displays the archives of the blog.

  • Trend Mini Gallery

    A mini gallery you can use to show a grid of images in the footer or sidebar widget area.

  • Trend Popular/Latest Posts

    You can show a selectable number of posts from the blog here.

  • Trend Latest Projects

    You can show a selectable number of Portfolio projects here.

  • Trend Sidebar Vimeo Video

    Show a vimeo video with this widget. Note to only add the vimeo video id NOT the whole link to the video.

  • Trend Sidebar Youtube Video

    Show a youtube video with this widget. Note to only add the youtube video id NOT the whole link to the video.

  • Trend Twitter Feed

    A twitter feed widget. You can select the number of tweets you want to show.

  • Trend HTML Shortcodes

    Special Widget to show our own Shortcodes in the Sidebars as well.

Contact

The Contact is a little special here. You will find a "Contact" template in the selectbox on the right hand of your WP page edit screen. This will add a Google Map to the top of your Contact page. This is not necessary. You could also go with a normal "Content" page.

We used the Contact Form 7 plugin for our Contact page which is pretty easy to implement. We styled the Input fields in our CSS so they will match the theme style. Also we build an add-on that enables you to use shortcodes inside the form. First download it via the plugin backend and activate it.

Shortcodes & Styles

Shortcodes are available in the content section of each page or post. 

We have build in several shortcodes that all can be put in via the editor directly (list of shortcodes is at the end of this section) or via our build-in shortcode wizard.

The wizard is available in the Visual mode of the Editor. Like with all of our Themes look out for the theme icon in the button panel of the editor (Example AlterEgo here):

 

After clicking on the section you are interested in a popup with a webform will guide you through building the shortcode.

You can then edit by hand, copy&paste and so on. 

A hard to read overview over all shortcodes and it options :

Localization / Words

This theme comes with .mo and .po files and is ready for localization! Note that most of the steps that lead to a localized theme are not related to this theme specifically but are general information.

  • I would highly recommend that you install the free plugin "CodeStyling Localization" which makes localizing and adding new languages very comfortable.

  • On this page you can get a link for your language that allows you to download files that turn your WordPress installation (the non theme files) into your desired language. You need the .mo and .po files for your language (Example link for german version here)
  • There core WordPress .mo and .po files you downloaded go into your wp-content/languages folder.
  • In the root of your WordPress installation you will find the wp-config.php file. Open it and change define ('WPLANG', ''); to reflect your country language code, e.g.: define ('WPLANG', 'de_DE');.
  • When going to the localization plugin under...
    » Tools » Localization
    ...you will see the Trend theme when you scroll down. You can customize the available languages (Deutsch/English) or add your own language.
  • After adding a new language, hit "Rescan" so the plugin can go through the theme files and get all the custom language fields.
  • Clicking "Edit" you can then translate those language fields.

  • Via...

    » Settings » General

    ...you can also set the themes "Time Format" which is used throughout the theme.
